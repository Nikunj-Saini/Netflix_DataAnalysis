# Netflix Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset to extract meaningful insights about content availability, trends, and distribution on the Netflix platform.

---

## Dataset Information
- Source: Netflix Movies and TV Shows Dataset
- File format: CSV
- Description: Contains information about movies and TV shows available on Netflix.

### Key Columns
- title  
- type (Movie / TV Show)  
- release_year  
- rating  
- duration  
- country  
- listed_in (Genre)  
- date_added  

---

## Tools and Libraries Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

---

## Analysis Performed
- Distribution of Movies vs TV Shows  
- Year-wise content addition trend  
- Country-wise content analysis  
- Rating distribution  
- Genre analysis  
- Movie duration and TV show season analysis  
- Handling missing and inconsistent values  

---

## Key Insights
- Netflix has a higher number of movies compared to TV shows  
- Most content was added after 2016  
- The United States and India contribute the largest share of content  
- TV-MA and TV-14 are the most frequent ratings  
- Drama and International content dominate the catalog
